## LuaJIT but with:
- continue
- Compound Operators
- Bitwise Operators
- "Parse-Time" Lambda Strings ``` `(x) x + 1 => x + 2` ```
- "Parse-Time" Hash Constants ``` `{ELF}TestHash` ```
- Windows Patch for UTF16 File Names
- Self Iterating Objects ``` for thing in table do ```

Only x86 will work
